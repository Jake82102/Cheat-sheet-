CSS Selectors and Specifcity

In CSS, Selectors are used to target HTML elements on web pages so they can be styled.

Selectors

Selector Tag Description
Type h1 {} The type selector matches the type of the element in the HTML document.
Universal * {} The universal selector slects all elements of any type.
Class < p class="example">...< /p > When you want to style an element. In CSS .example
ID < h1 id="example">...< /h1 > When you want to uniquely style an element. In CSS #example
Attritbute type[attribute*=value] Attribute is used to select HTML elements that already have attributes like img
Pseudo-class p:example A pseudo-class is used for when you click on a link thats blue and when you come back it's purple, or when youre filling out a form and the submit button is grayed out but when your done the button has color and is active.

Specifcity

Specificity is the order by which the browser decides which CSS styles will be displayed.

Selector Description
ID ID is the most specfic selector. Will only change element.
Class This is the second most specfic. Will only change the elements woth the same class.
type This is the least specfic and will effect every element attached to it.
